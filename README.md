# SmartResumeAnalyzer.AI

## 📌 Overview
A web-based application that analyzes resumes using NLP and provides insights on skills and job relevance.

## 🚀 Features
* Resume analysis
* Skill extraction
* Keyword matching
* Improvement suggestions

## 🛠️ Tech Stack
Python, Flask, HTML, CSS, NLP

## ▶️ Run Locally
```bash
git clone https://github.com/subhaprayashbisoi-rgb/SmartResumeAnalyzer.AI.git
cd SmartResumeAnalyzer.AI
pip install flask
python app.py
```

Open: http://127.0.0.1:5000/

## 🎯 Use Case
Helps students and job seekers improve their resumes.
